## Development Setup Local

1. Clone repository
```
git clone git@github.com:AlexGalhardo/learning-json-rpc.git
```

2. Install dependencies
```
bun install
```

3. Create .env
```
cp .env.example .env
```

4. Up JSON-RPC server
```
bun run server
```

5. Run client request
```
bun run client
```
